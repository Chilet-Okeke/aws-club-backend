This repo consists of a simple backend to test the connection between a frontend service on an 
AWS S3 bucket and a backend service on an AWS ec2 instance for the Uniuyo Cloud Club aws-deployment session.

For more details please read the full session docs at https://github.com/Chilet-Okeke/aws-deployment

Disclaimer: Please note that this is a sample backend service and does not fully follow software development and devops 
best practices particularly in code structure and security architecture.
